# **App Name**: SheetSurfer

## Core Features:

- Data Fetching: Fetch data from the published Google Sheet via its URL.
- Data Display: Display the fetched Google Sheet data in a user-friendly format.
- Custom Skin: Apply a custom visual theme to the displayed data.
- Landing Page: Design a basic landing page with information and access to the Google Sheet data.

## Style Guidelines:

- Background color: Almost-white background (#F2F2F2) to provide a clean and neutral canvas, ensuring readability and focus on the content.
- Primary color: Dark gray (#333333) for text elements and key interactive components, offering a strong contrast against the light background and conveying sophistication.
- Accent color: Light gray (#D3D3D3) for subtle highlights, dividers, and secondary elements, maintaining the minimalist aesthetic and providing visual separation.
- Headline font: 'Orbitron' (sans-serif) for headings, giving a modern and slightly futuristic feel. Note: currently only Google Fonts are supported.
- Body font: 'Kode Mono' (monospace) for body text, providing a clean and readable experience. Note: currently only Google Fonts are supported.
- Clean and minimalist layout with a focus on readability and ease of navigation.
